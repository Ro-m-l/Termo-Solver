# Termo-Solver
Um trecho de código em python feito como um exemplo de engenharia reversa para decodificar as respostas do jogo termo (https://term.ooo/), utilizando apenas informações obtidas no código frontend do website.

Limitação: Apesar de apresentar o funcionamento de modo simples e entendível, mudanças futuras de código não estão sendo consideradas. Utilizar Playwright como fallback seria uma opção para ter mais consistência e facilidade em encontrar a resposta, mas menos eficiente e continuaria lidando com possíveis mudanças e obfuscações do código.
