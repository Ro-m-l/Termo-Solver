## Considerações futuras
Apesar de apresentar o funcionamento de modo simples e entendível, mudanças futuras de código não são consideradas. Utilizar Playwright como fallback seria uma opção para ter mais consistência e facilidade em encontrar a resposta, mas menos eficiente e, com possíveis mudanças e obfuscações do código, sujeito a depender da interpretação do usuário em relação ao output.
